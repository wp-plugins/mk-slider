== Description ==
MK Slider :  MK Slider is a Wordpress slider with stunning effects and very easy to use. You can create any number of sliders in a matter of seconds without coding & image editing. It also support shortcode for adding it to posts or pages if you want to. MK Slider also provide the slider settings to each sliders created individually. Also there is a widget if you want to add the MK Slider to the sidebar of your wordpress blog.

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Now you can view MK Slider Tab added to  your wordpress admin under the settings tab.

4. Click 'Add New' to create a new slider. The shortcode and the template tag for the newly created slider will be visible in the right sidebar on the create new page. You can use them to either add the slider to posts/pages or in the template.


== Frequently Asked Questions ==
= How to add the slider to the template = 
You can get the template tag for the slider from the right sidebar of the edit slider window.
= How to add the slider to the post/page = 
You can get the shortcode for the slider from the right sidebar of the edit slider window.
= How many sliders can I create = 
You can create any number of slider you wan using MK Slider
= How many I add MK Slider to sidebar =
