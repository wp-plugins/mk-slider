== Description ==
MK Slider :  MK Slider is a Wordpress slider with stunning effects and very easy to use. You can create any number of sliders in a matter of seconds without coding & image editing. It also support shortcode for adding it to posts or pages if you want to. MK Slider also provide the slider settings to each sliders created individually.

Created By: Manoj Rana
Email : manoj.rana91986@gmail.com

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Now you can view MK Slider Tab added to  your wordpress admin under the settings tab.

4. Click 'Add New' to create a new slider. The shortcode and the template tag for the newly created slider will be visible in the right sidebar on the create new page. You can use them to either add the slider to posts/pages or in the template.


== Frequently Asked Questions ==
